# Test Setup

This is a react test setup, that needs to be cloned in your system to proceed further
Please follow below steps/instructions for the assessment  -

### Instructions

+ Basic syntax search is allowed, however please refrain from seeking solutions directly from
online LLMs.
+ You will have **30mins** to complete the assessment.
+ Correctness of solutions will be judged on approach chosen by the candidate and functional
correctness rather than rigid implementation expectations.

### Steps

+ Clone the repo.
+ Install necessary packages.
+ Refer to Tasks folder for your assessment.
+ Each subsequent step builds on previous task's code.
+ Create a branch with main as the base, Name it **candidateName | FE Test**
